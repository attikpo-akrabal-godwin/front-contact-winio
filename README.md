# Nuxt 3  : front de l'app de contact 

  



  

## lancer 

  


  

```bash

# npm

npm  install

```

  

## lancer le serveur de développement
 le serveur backend doit être lancer sur http://localhost:4000/api .
 
lancer votre serveur sur : `http://localhost:3000`:

  

```bash

# npm

npm  run  dev

  

# pnpm

pnpm  run  dev

  

# yarn

yarn  dev

  

# bun

bun  run  dev

```
